This privacy policy applies to the HeloChecklist app (hereby referred to as "Application") for mobile devices, created by ercanay ceran (hereby referred to as "Service Provider") as a Freemium service. This service is intended for use "AS IS".

HeloChecklist is a professional aviation utility application designed for pilots operating the Black Hawk helicopter. It provides a comprehensive digital checklist system covering all phases of flight operations, including pre-flight inspection, engine start, taxi, takeoff, cruise, approach, landing, shutdown, and emergency procedures.

Core Features

  • Flight Checklists. A complete set of normal and emergency procedure checklists. Items can be checked off individually, with support for undo, phase reset, and automatic phase progression.
  • Flight Logging. Records engine-on and engine-off times to calculate flight duration. Log entries are stored locally on the user's device.
  • Navigation Assistance. Uses the device's GPS to calculate bearing, distance, and estimated time en route to a user-defined destination. Location data is used solely for in-app calculations and is never transmitted to external servers.
  • Weather Information. Retrieves METAR and TAF reports via a secure proxy server. Weather data is fetched on demand and not stored beyond the current session.
  • Offline Maps. Downloads and stores map tiles locally to enable offline map display. Satellite imagery is sourced from publicly available NASA and ESRI tile services.
  • Navigation Aid Database. Maintains a local database of airports, heliports, VORs, NDBs, and other navaids sourced from the OpenAIP aviation data service, refreshed per AIRAC cycle.
  • Apple Watch Integration. Communicates checklist state and flight data to a paired Apple Watch using Apple's Watch Connectivity framework. No data leaves the user's device pair.
  • Siri and Shortcuts. Supports Apple Shortcuts and Siri voice commands for opening checklist phases, resetting flight state, and querying logged flight hours.

All user data — including flight logs, checklist state, pilot preferences, and cached map and navigation data — is stored exclusively on the user's device. The Application does not operate user accounts, does not collect personally identifiable information, and does not share user data with third parties except as necessary to retrieve weather information and navigation aid data from the services described above.
