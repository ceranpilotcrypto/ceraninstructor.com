Privacy Policy

Effective date: 2026-05-16

This privacy policy applies to the HeloChecklist app (hereby referred to as "Application") for mobile devices and Apple Watch, created by ercanay ceran (hereby referred to as "Service Provider") as a freemium service. This service is intended for use "AS IS".

HeloChecklist is a professional aviation utility application designed for pilots operating the Black Hawk helicopter. It provides a comprehensive digital checklist system covering all phases of flight operations, including pre-flight inspection, engine start, taxi, takeoff, cruise, approach, landing, shutdown, and emergency procedures.

Core Features

  • Flight Checklists. A complete set of normal and emergency procedure checklists. Items can be checked off individually, with support for undo, phase reset, and automatic phase progression.
  • Flight Logging. Records engine-on and engine-off times to calculate flight duration. Log entries including destination, distance, bearing, wind conditions, and mission debrief information are stored locally on the user's device. Log entries may be exported as a PDF document or shared as plain text by the user using the device's standard sharing features. This export is entirely user-initiated and at the user's discretion.
  • Navigation Assistance. Uses the device's GPS to calculate bearing, distance, and estimated time en route to a user-defined destination. The Application collects precise real-time location data solely for in-app navigation calculations. This data is processed entirely on the user's device and is never transmitted to external servers.
  • Weather Information. Retrieves METAR and TAF reports via a secure Cloudflare proxy server operated by the Service Provider. Only the requested airport identifier is transmitted. Weather data is fetched on demand and is not stored beyond the current session.
  • Offline Maps. Downloads and stores map tile imagery locally to enable offline map display. Satellite imagery is sourced from NASA and ESRI tile services. When an internet connection is available, map label tiles may be fetched from Stadia Maps (stadiamaps.com) solely for display purposes. No user data is transmitted to Stadia Maps.
  • Navigation Aid Database. Maintains a local database of airports, heliports, VORs, NDBs, and other navigation aids sourced from the OpenAIP aviation data service, refreshed per AIRAC cycle. Queries to the OpenAIP service transmit only geographic bounding coordinates, not any personal data.
  • Crew Sync. Allows two pilots to share real-time checklist state, engine timer data, and mission information over a direct peer-to-peer Bluetooth or local WiFi connection using Apple's Multipeer Connectivity framework. No data is routed through external servers. Data is transmitted only between the two participating devices during an active session and only with the explicit consent of both pilots. The transmitted data includes checklist item states, engine start and stop times, and mission debrief fields entered by the initiating pilot.
  • Apple Watch Integration. Communicates checklist state, flight data, navigation data, and voice-read checklist items to a paired Apple Watch using Apple's Watch Connectivity framework and on-device speech synthesis. No data leaves the user's device pair.
  • Siri and Shortcuts. Supports Apple Shortcuts and Siri voice commands for opening checklist phases, resetting flight state, and querying logged flight hours. No data is transmitted to third parties through this integration beyond Apple's standard Siri processing.
  • Barometric Sensor. Uses the device's built-in barometric altimeter to calculate QNH and density altitude. Sensor data is processed entirely on-device and is never stored or transmitted.
  • In-App Purchases and Subscriptions. The Application offers optional premium features through a monthly or annual auto-renewing subscription purchased via Apple's App Store. Subscription management, billing, and receipt validation are handled exclusively by Apple Inc. The Service Provider does not collect or store payment information. Transaction metadata is processed by Apple in accordance with Apple's Privacy Policy.

What information does the Application obtain and how is it used?

The Application collects precise GPS location data from your device solely to provide in-app navigation features. This data is never transmitted to external servers and is not stored beyond your current session. All other user data — including flight logs, checklist state, pilot preferences, mission debrief records, and cached map and navigation data — is stored exclusively on your device.

The Application does not operate user accounts and does not collect personally identifiable information beyond what you voluntarily enter (such as pilot name, unit designation, callsign, and crew information in mission debrief records). This information is stored locally on your device only.

Do third parties see and/or have access to information obtained by the Application?

The Application makes network requests to the following third-party services solely to provide stated features:

  • Cloudflare (Service Provider proxy) — weather data retrieval. Only airport identifiers are transmitted.
  • OpenAIP — navigation aid data. Only geographic coordinates are transmitted.
  • Stadia Maps — online map label tiles. No user data is transmitted.
  • Apple Inc. — subscription and in-app purchase processing, Siri integration, Watch Connectivity.

No personal data, flight logs, location history, or mission information is shared with any third party.

Opt-Out Rights

You can stop all data collection by uninstalling the Application using the standard uninstall processes available on your mobile device or via the App Store. Subscription cancellation is managed through your Apple ID settings.

Security

All network communications use HTTPS/TLS encryption. Crew Sync peer-to-peer connections use end-to-end encryption provided by Apple's Multipeer Connectivity framework. Sensitive configuration values are obfuscated in the application binary. No user data is stored on external servers.
